# FalconStats (WORK IN PROGRESS)



A system created for a bachelor's thesis. Used to display statistics for baseball team players.

System created with :
- Spring boot
- Hibernate
- Bootstrap
- Thymeleaf
- IntelliJ IDE

Statistics we can see :
- Batting statistics
- Baserunning statistics
- Pitching statistics
